## 🎯 こんな人に読んでほしい

```cpp
// こんな経験、ありませんか？

void* ptr = malloc(sizeof(int));
// ...100行後...
free(ptr);
// ...さらに50行後...
*ptr = 42;  // 💀 Use after free

std::vector<int>& get_ref() {
    std::vector<int> v = {1, 2, 3};
    return v;  // 💀 Dangling reference
}

// Rustなら、これらは**コンパイル時**にエラーになる。
```

- C/C++での開発経験がある
- Rustに興味があるが、所有権で挫折した
- 「borrow checker」に何度も怒られた
- 物語形式で楽しく学びたい
